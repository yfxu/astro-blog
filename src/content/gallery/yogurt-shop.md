This was behind the counter of some random yogurt drink shop we went to in China. It seems like yogurt drinks are pretty popular in China. Usual toppings include things like mochi, purple rice, oats, etc. It sounds pretty healthy if not for all the extra sugar that they add to make it taste good...

It was very blue inside the store, as you can maybe tell from the very solidly-coloured wall.
